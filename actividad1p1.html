INFORME

HTML: es una herramienta que se utiliza para crear paginas web en internt. Para ello utilizamos sublime text es una herramienta donde construimos paso por paso la pagina alli se utilizan etiquetas como: Body; Para el contenido, Head: Para informacion del documento, Div: Mover cosas en la pantalla, a: hipervinculos, br: Saltos en linea, Img: Para imagenes. Siempre se debe tener encuenta que cuando vamos a colocar una etiqueta hay que colocar el simbolo <(menor que)y se debe de termina con este >(mayor que)

CONTENEDORES

Hay diversos tipos de contenedores, en los que se pueden modificar su forma, tamaño, color y diversidad de cosas. Que son los contenedores; son lugares donde podemos guardar informacion concreta y mas ordenada en cual quier lugar de una pagina web. Para realizar un contenedor debemos de colocar la etiqueta "div" tambien lo podemos editar con otras etiquetas como "background" para colocarle color a los contenedores, "Font-family" para cambiar la letra de el texto, "Font-size" para cambiar el tamaño de el texto, etc.
